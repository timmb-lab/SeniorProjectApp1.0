# Automation Self-Improvement Protocol

This protocol applies only to Functionality UX Upgrade (`functionality-ux-upgrade-hourly`).

Use `docs/functionality-ux-growth-ladder.md`, `docs/functionality-ux-growth-ledger.md`, and `automation/state/functionality-ux-growth-state.json` to choose, record, and avoid repeating product-readiness work.

Validate prompt, ladder, ledger, or state changes with:

```powershell
npm run verify:functionality-ux-automation
```

Do not add, revive, or document another active automation without explicit future instruction.
