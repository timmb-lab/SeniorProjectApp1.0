# Archived Files

This folder holds retired or legacy artifacts that are no longer active app,
guide, deploy, or build surfaces.

- `old/stakeholder-options/` contains historical Titan Blend and Back To Basics
  review outputs.
- `legacy-redirects/` contains old root redirect shells that used to point back
  to the guide home.
- `scripts/` contains retired generators kept only for historical reference.

Active source, generated output, tests, migrations, and documentation stay at
the repo root paths used by the current app and guide workflows.
